Reminisce
Tutorial Documentation Source code

Reminisce is a free, open-source Rust library for detecting and interacting with gamepads.

Supported Platforms

  • Linux (using joystick API or SDL)
  • Windows (using XInput or SDL)
  • Emscripten (when support arrives it should work out of the box)

Features

  • Events for axis and button movement
  • Every joystick can have its events processed internally into its state by using the StatefulJoystick API
  • Iterator API for easy event detection
